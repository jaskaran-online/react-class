CSS Grid Tutorial

1. Introduction to CSS Grid

CSS Grid is a two-dimensional layout system that allows you to create complex web layouts with rows and columns. It's perfect for both overall page layout and smaller component layout.

2. Basic Grid Layout

Create a simple grid using display: grid and define columns using grid-template-columns.

display: grid;
grid-template-columns: repeat(3, 1fr);
gap: 10px;

Key Grid Properties

Container Properties:

  • display: grid - Defines a grid container
  • grid-template-columns - Defines the columns of the grid
  • grid-template-rows - Defines the rows of the grid
  • gap - Sets spacing between grid items
  • grid-template-areas - Defines named grid areas
  • justify-content - Aligns grid along the row axis
  • align-content - Aligns grid along the column axis

Item Properties:

  • grid-column - Specifies item's location by column
  • grid-row - Specifies item's location by row
  • grid-area - Places item in named grid area
  • justify-self - Aligns item within its cell (horizontal)
  • align-self - Aligns item within its cell (vertical)

Grid Units and Functions

  • fr - Fractional unit, shares available space
  • repeat() - Repeats columns or rows
  • minmax() - Sets minimum and maximum sizes
  • auto-fit - Fits as many items as possible
  • auto-fill - Similar to auto-fit but keeps empty tracks

Navigation Menu with Dropdown - Documentation

1. HTML Structure

Basic Hierarchy:

nav.nav-menu
    └── ul.nav-list
        └── li.nav-item
            ├── a.nav-link
            └── ul.dropdown
                └── li.dropdown-item
                    ├── a.nav-link
                    └── ul.subdropdown
                        └── li.dropdown-item
                            └── a.nav-link

2. CSS Components

Reset Styles:

Applied to: All elements (*)

  • margin: 0 - Removes default margins
  • padding: 0 - Removes default padding
  • box-sizing: border-box - Includes padding in width calculations

Main Navigation (.nav-menu):

Applied to: <nav> element

  • background-color: #333 - Dark background
  • padding: 1rem - Inner spacing

Navigation List (.nav-list):

Applied to: Primary <ul>

  • list-style: none - Removes bullets
  • display: flex - Horizontal layout
  • gap: 2rem - Space between items

Navigation Items (.nav-item):

Applied to: All <li> elements

  • position: relative - Creates positioning context

Dropdown Menus (.dropdown, .subdropdown):

Applied to: Secondary and tertiary <ul> elements

  • position: absolute - Removes from normal flow
  • background-color: #444 - Dropdown background
  • min-width: 200px - Minimum width
  • display: none - Hidden by default

3. Positioning

Dropdown Positioning:

.dropdown:

  • top: 100% - Appears below parent
  • left: 0 - Aligned with parent's left edge

.subdropdown:

  • top: 0 - Aligned with parent's top
  • left: 100% - Appears to the right

4. Interactive Features

Hover Effects:

  • Shows dropdowns on parent hover
  • Changes background color on hover
  • Uses direct child selector (>)

Visual Indicators:

  • ▼ arrow for main dropdown items
  • ▶ arrow for subdropdown items
  • Added using ::after pseudo-element

5. Key Features

  • Pure CSS solution - No JavaScript required
  • Supports multiple levels of dropdowns
  • Hover-based interaction
  • Visual indicators for navigation
  • Responsive and maintainable structure

6. Best Practices

  • Keep dropdown levels to a maximum of 2-3 for better UX
  • Ensure sufficient contrast for text visibility
  • Maintain consistent spacing and padding
  • Use semantic HTML structure
  • Consider mobile responsiveness

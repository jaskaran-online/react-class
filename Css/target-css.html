Frequently Asked Questions

  • What is CSS?
  • What is the :target pseudo-class?
  • How does the :target pseudo-class work?

What is CSS?

CSS stands for Cascading Style Sheets. It is a styling language used to describe the presentation of a document written in HTML or XML.

What is the :target pseudo-class?

The :target pseudo-class in CSS is used to style the target element of a URI containing a fragment identifier.

How does the :target pseudo-class work?

When a URI's fragment identifier points to a target element in the document, the :target pseudo-class is applied to that element. This allows you to style the targeted element differently.

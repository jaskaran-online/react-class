CSS Animation Practice Tasks

Task 1: Hover Effect

Learn how to create a basic hover effect using transitions:

  • Change background color on hover
  • Scale the element up on hover
  • Use transition for smooth animation

Task 2: Multiple Transforms

Practice combining multiple transforms:

  • Rotate the element
  • Scale it up
  • Move it horizontally

Task 3: Bouncing Ball

Create a bouncing animation using @keyframes:

  • Use translateY to move the ball
  • Make it bounce infinitely
  • Add easing for natural movement

Task 4: Loading Animation

Create a loading animation with multiple elements:

  • Animate multiple dots
  • Use animation delays
  • Create a sequence effect

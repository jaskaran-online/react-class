CSS Positions and Z-Index Exercise

Task:

Create four boxes with different positioning properties and z-index values to understand how they interact with each other.

Relative
Absolute
Fixed
Sticky

Solution Explanation:

  1. Box 1 (Red): Uses relative positioning, moved slightly from its normal position
  2. Box 2 (Green): Uses absolute positioning, positioned relative to its closest positioned ancestor
  3. Box 3 (Blue): Uses fixed positioning, stays in place even when scrolling
  4. Box 4 (Purple): Uses sticky positioning, behaves like relative until the scroll threshold

Key CSS Properties Used:

.box {
    position: relative|absolute|fixed|sticky;
    top: value;
    left: value;
    z-index: value;
}

Important Notes:

  • Z-index only works on positioned elements (position value other than static)
  • Fixed positioning is relative to the viewport
  • Sticky positioning is a hybrid of relative and fixed
  • Higher z-index values appear on top of lower values

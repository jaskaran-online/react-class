Frequently Asked Questions

What is CSS?

CSS stands for Cascading Style Sheets. It's a styling language used to describe the presentation of a document written in HTML or XML.

What is the :target pseudo-selector?

The :target pseudo-selector in CSS is used to style the target element of a URL containing a fragment identifier.

How does the :target selector work?

When a URL's fragment identifier matches the ID of an element in the document, the :target selector applies styles to that element.

Flexbox Tutorial

Basic Flexbox Concepts

Flexbox is a one-dimensional layout method for arranging items in rows or columns. Items flex (expand) to fill additional space or shrink to fit into smaller spaces.

Flex Item Properties

Individual flex items can be customized using flex-grow, flex-shrink, and flex-basis.

flex-grow: 1
flex-grow: 2
basis: 100px

Detailed Flex Item Properties

1. Flex Grow

flex-grow determines how much space an item takes up relative to other items. Default value is 0.

flex-grow: 1
flex-grow: 2
flex-grow: 1

2. Flex Shrink

flex-shrink determines how items shrink when there isn't enough space. Default value is 1.

shrink: 1
shrink: 2
shrink: 0

3. Flex Basis

flex-basis sets the initial main size of an item. Can be a length (px, em) or auto.

basis: 100px
basis: 200px
basis: auto

4. Combined Properties

The flex shorthand combines grow, shrink, and basis: flex: grow shrink basis

flex: 1 1 auto
flex: 2 1 200px
flex: 0 0 150px

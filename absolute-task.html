Absolute Positioning Exercise - Social Media Profile Card

Task:

Create a social media profile card with the following requirements:

  1. Create a profile card with a header background
  2. Add a circular profile picture that overlaps the header and content area
  3. Add a verification badge that should be positioned at the bottom-right of the profile picture

Solution Explanation:

  1. Container Setup: The main container uses relative positioning to establish a positioning context
  2. Profile Picture: Uses absolute positioning to overlap the header and content areas
  3. Verification Badge: Uses absolute positioning relative to the profile picture

Key CSS Properties Used:

.demo-container {
    position: relative;
}

.profile-picture {
    position: absolute;
    top: 40px;
    left: 90px;
}

.badge {
    position: absolute;
    bottom: 10px;
    right: 10px;
}

Important Notes:

  • The parent container must have position: relative for absolute positioning to work properly
  • Absolute positioning removes elements from the normal document flow
  • Coordinates (top, left, bottom, right) are relative to the nearest positioned ancestor
  • Use padding and margins carefully when working with absolute positioning

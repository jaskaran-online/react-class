Media Query Tutorial

What are Media Queries?

Media queries allow you to apply CSS styles based on various device characteristics and parameters like screen size, device orientation, or display type. They are fundamental for creating responsive web designs.

Basic Syntax

@media media-type and (media-feature) { /* CSS rules */ }

Common media types include:

  • all - All devices
  • screen - Computer screens, tablets, phones
  • print - Printers and print preview

Common Breakpoints

  • Mobile: @media (max-width: 480px) { }
  • Tablet: @media (max-width: 768px) { }
  • Laptop: @media (max-width: 1024px) { }
  • Desktop: @media (max-width: 1200px) { }

Best Practices

  • Start with mobile-first approach
  • Use relative units (em, rem, %)
  • Test across multiple devices
  • Keep breakpoints consistent
  • Don't base breakpoints on specific devices
